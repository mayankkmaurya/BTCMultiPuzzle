# Client Downloads

Download the latest mining clients from:

**[btcmultipuzzle.com/join/#download](https://btcmultipuzzle.com/join/#download)**

Or use the direct links:

- [Windows 64-bit](https://btcmultipuzzle.com/join/clients/puzzle-client-win64.zip)
- [Ubuntu 24.04](https://btcmultipuzzle.com/join/clients/puzzle-client-ubuntu-24.tgz)
- [Ubuntu 22.04](https://btcmultipuzzle.com/join/clients/puzzle-client-ubuntu-22.tgz)

## Why aren't the binaries in this repo?

GitHub has file size limits for repositories. The client binaries are hosted on our main server for faster downloads and easier updates.
