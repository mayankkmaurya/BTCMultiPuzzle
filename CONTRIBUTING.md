# Contributing to BTC Multi Puzzle

Thanks for your interest in contributing!

## Ways to Contribute

### 1. Run a Miner
The easiest way to contribute is to run the mining client. Every GPU helps increase our collective odds.

### 2. Spread the Word
Share the project with others who might be interested. More miners = better odds for everyone.

### 3. Report Issues
Found a bug or have a suggestion? Open an issue on GitHub or message us on [Telegram](https://t.me/btcmultipuzzle).

### 4. Documentation
Help improve the docs, fix typos, or add translations.

## Community Guidelines

- Be respectful to other community members
- Keep discussions on-topic
- No spam or self-promotion
- Report bugs constructively

## Questions?

Join our [Telegram group](https://t.me/btcmultipuzzle) for real-time support and discussion.
